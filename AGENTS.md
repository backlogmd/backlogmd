# Agents

This project uses [backlogmd](https://www.backlogmd.com) (`.backlogmd/`) for work tracking.

**Always use the `/backlogmd` skill to interact with the backlog.**

## Workflow

- **Before planning**: Check the backlog for existing items and tasks.
- **When planning**: Create items and tasks in the backlog.
- **When starting work**: Mark tasks as `in-progress`.
- **When completing work**: Mark tasks as `done`.
